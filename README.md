# 🚀 TK Nexus - AI-Powered Business Operating System

Complete ready-made SaaS application for growing small businesses using AI.

## Features

✅ Lead Generation from Google Maps, Instagram, LinkedIn
✅ Automated Outreach (Email, WhatsApp, Instagram, SMS)
✅ AI Website Builder
✅ Social Media Content Generator
✅ Analytics Dashboard
✅ AI Assistant

## Quick Start

### Frontend
```bash
cd frontend
npm install
npm run dev
# http://localhost:3000
```

### Backend
```bash
cd backend
npm install
npm run dev
# http://localhost:3001
```

## Tech Stack

**Frontend:** Next.js, React, TailwindCSS, TypeScript
**Backend:** Node.js, Express, TypeScript
